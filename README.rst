`GitHub Pages`_ site for a custom JetBrains IDEA `plugin repository`_ for
`github.com/mdklatt`_.

Deployment
**********

Pushes to ``main`` are automatically deployed to `mdklatt.github.io/idea-plugin-repo`_
by `GitHub Actions`_.
